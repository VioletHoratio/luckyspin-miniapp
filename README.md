# LuckySpin

LuckySpin is a Base Mini App for rapid onchain lottery spins with transparent reveal and jackpot pool flow.

## Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- wagmi + viem
- lucide-react
- framer-motion

## Environment

Copy `.env.example` to `.env.local` if you want to override public builder configuration locally.

## Scripts

- `npm run dev`
- `npm run build`
- `npm run start`
